# KeyBank Fraud Case Analysis

This project presents an ethical and financial analysis of the KeyBank PPP fraud case. It examines the role of internal control failures, fraudulent loan activity, regulatory consequences, and the broader importance of ethics in banking and finance.

## Main topics
- Ethics in banking and finance
- PPP loan fraud
- Internal control and compliance failures
- Civil and criminal consequences
- Key lessons for financial institutions

## Key takeaway
This case shows that identifying red flags is not enough. Financial institutions must respond quickly, strengthen internal controls, and uphold ethical standards to protect public trust.
